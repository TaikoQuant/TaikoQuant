# TaikoQuant

太鼓シミュです。

## 今後対応する予定の機能
- [ ] 連打,風船
- [ ] HBSCROLL
- [ ] BMSCROLL


## ビルド方法
`dotnet build -c Release`

## 注意
ビルドした際に起動するときは、 **`bass.dll`** が必要です。
以下の公式サイトよりダウンロードし、実行ファイルと同じディレクトリに配置してください。

[bass24.zip をダウンロード](https://www.un4seen.com/files/bass24.zip)


tjaを読み込むライブラリを使用しております。
[TJAChartCore](https://github.com/touhourenren/TJAChartCore)
